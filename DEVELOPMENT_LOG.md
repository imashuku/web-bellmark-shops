# ウェブベルマーク掲載ショップ一覧 開発経緯

## 概要
ウェブベルマークの掲載ショップ一覧サイトのUI/UX改善プロジェクト。
ユーザーが「こんなに提携ショップがあるなら使ってみようかな」と感じるサイトを目指した。

## 開発期間
2025年12月14日

## 主な改善ポイント

### 1. ヒーローセクションの改修
**課題**: ショップ数の多さが伝わらない

**解決策**:
- 店舗数「235」を大きく目立つように表示
- 「いつものネットショッピングが、学校への支援に」というキャッチコピー追加
- 日付表示を動的に生成（`CURRENT_DATE`変数で管理）

**制約事項**:
- 紙のベルマークと直接比較してはいけない（ウェブベルマークは紙のベルマークのインフラに依存しているため）

### 2. 全体デザインのリデザイン（1回目）
**課題**: 「ダサい」「AIっぽい」デザイン

**変更内容**:
- ヘッダー: 紫グラデーション → ネイビー（#1e3a5f）
- 背景: グレー → 白
- カード: ドロップシャドウ → フラットなボーダー
- バッジ: 黄色 → ネイビー
- 絵文字を全て削除（AIっぽさを払拭）
- 日本語フォントスタック最適化
- コンテナ幅: 1200px → 960px

### 3. 説明文の切り詰め処理
**課題**: 長い説明文が途中で切れて不親切

**検討した案**:
- 案A: JS-based overflow detection
- 案B: Simple 3-line truncation with "..."
- 案E: Fade-out gradient

**採用**: 案B（シンプルな3行切り詰め）
- `-webkit-line-clamp: 3`を使用
- フェードアウトは「表示できているのにフェードする」違和感があったため却下

### 4. 高還元ショップの優先表示
**課題**: どのショップが高還元かわからない

**実装内容**:
- 還元率（%）TOP10と還元点数（点）TOP10を2列で表示
- 各ショップカードの還元率を赤色（#c41e3a）で大きく強調
- `.rate-number`と`.rate-unit`クラスで数字と単位を分離

### 5. Bento UIへのリデザイン（最終形）
**課題**: より現代的でインパクトのあるデザインに

**実装内容**:
```
┌──────────────┬──────────────┐
│              │  還元率TOP5   │
│   235        ├──────────────┤
│  提携ショップ  │  還元点数TOP5  │
├──────────────┼───────┬──────┤
│   検索       │  18   │  6   │
│   ボックス    │カテゴリ│おすすめ│
└──────────────┴───────┴──────┘
```

**特徴**:
- 4列グリッドの異なるサイズカード配置
- 角丸20pxの大きいカードデザイン
- 薄いグレー背景（#f5f5f7）
- ホバー時の浮き上がりアニメーション
- TOP10 → TOP5に変更（Bentoカードに収まるように）

### カテゴリ順序の最適化

2. 教育用品（学校用品）
3. 教育・学習サービス（Z会、進研ゼミ等）
4. 本・音楽・ゲーム
5. キッズ・ベビー・おもちゃ

## 学んだこと・注意点
1. **AIっぽさの回避**: 絵文字の多用、過度なグラデーション、Bootstrap風のデザインは避ける
2. **ユーザー視点**: 「紙のベルマークより優れている」という比較は避け、「補完的」な立場を維持
3. **動的変数の活用**: 毎月変わる情報は変数化して更新を簡略化
4. **フィードバックの反映**: ユーザーの「違和感」を丁寧に聞き取り、代替案を提示
