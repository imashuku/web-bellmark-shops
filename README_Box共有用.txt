ウェブベルマーク 掲載ショップ一覧
==========================================

【オンラインで閲覧】（推奨）
以下のURLにアクセスすると、常に最新版をご覧いただけます：
https://imashuku.github.io/web-bellmark-shops/

【オフラインで閲覧】
このフォルダ内の index.html をダウンロードして、
ブラウザで開いてください。

【印刷・PDF出力】
上記URLまたはHTMLファイルを開き、
ブラウザの印刷機能（Ctrl+P / Cmd+P）から
PDFとして保存できます。
